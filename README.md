# Flask Blog

A **simple blogging application** built with **Flask, HTML, and CSS**. Users can create, edit, and delete blog posts easily.

## 🚀 Features
- **User Authentication** – Register, login, and logout functionality.
- **Create & Manage Posts** – Users can add, edit, and delete blog posts.
- **Database Support** – Uses PostgreSQL for database management.
- **Responsive UI** – Styled with HTML & CSS.

## 🛠 Tech Stack
- **Backend:** Flask, SQLAlchemy
- **Frontend:** HTML, CSS
- **Database:** PostgreSQL
